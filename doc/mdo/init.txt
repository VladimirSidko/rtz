Стуктура дистрибутива:
cfg - файлы конфигурации для usr-distr\cfg
cmd - файлы часто выполняемых команд для разработки
doc - файлы документации для разработки и исходные тексты результата для usr-distr\doc
etc - файлы настроек для разработки и прочая
img - файлы графические для usr-distr\img
inc - файлы включения для разработки (для исходников src)
iss - файлы для разработки инсталлятора usr-setup\
sql - файлы для разработки базы данных usr-distr\fdb и usr-distr\sql
src - файлы исходных текстов проекта
